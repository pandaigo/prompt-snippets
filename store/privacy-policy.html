Privacy Policy

Last updated: 2026-05-07

Prompt Snippets ("the Extension") is a Chrome browser extension that lets you save reusable prompt templates and insert them into AI chat services like ChatGPT, Claude, Gemini, and Perplexity with one click.

Data Collection

The Extension does not collect, transmit, or share any personal data or prompt content. Specifically:

  • Your prompt templates and variable values are stored only on your local device.
  • No prompts, variables, or page content are ever sent to any server.
  • No analytics, tracking, or telemetry is included.
  • No user accounts are required.
  • No cookies are set by the Extension.

Data Storage

All user data — prompt templates, categories, variable history, and settings — is stored locally on your device using the Chrome Storage API (chrome.storage.local). This data never leaves the device.

Permissions

The Extension requests the following permissions:

  • storage — To save your prompt templates and settings locally on the device.
  • activeTab — Required only when you click the extension icon or a snippet, to access the current tab's input field for one-click insertion. The Extension does not read the rest of the page.
  • scripting — Required to inject the insertion code into the active tab when you choose to insert a snippet. Code is only injected after you explicitly request insertion.
  • contextMenus — To add a "Prompt Snippets" right-click menu item on text input fields, so you can insert snippets directly from the right-click menu.
  • clipboardWrite — Used as a fallback when direct insertion is not possible (some pages block input access). The selected snippet is then copied to the clipboard so you can paste it manually with Ctrl+V.

Third-Party Services

The Extension uses ExtensionPay (extensionpay.com) to process one-time payments and verify Pro license status. ExtensionPay receives a randomly generated installation ID for the sole purpose of license verification. Your prompt templates and variable values are never transmitted to ExtensionPay or any third party. Payment itself is processed by Stripe via ExtensionPay; the Extension does not handle credit card data.
