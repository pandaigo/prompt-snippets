✦ Why Prompt Snippets?
If you use AI chatbots every day, you probably paste the same prompt over and over: blog outlines, code reviews, email rewrites, summaries. Prompt Snippets keeps your best prompts in one searchable library and inserts them into ChatGPT, Claude, Gemini, or Perplexity with a single click.

✦ Local-only by design
Unlike AIPRM and similar tools, Prompt Snippets does not show ads, does not require sign-in, and does not sync your prompts to any cloud. Built for writers, marketers, support agents, and anyone who handles confidential prompts.

✦ Variables that save real time
Mark dynamic parts with curly braces like {topic}, {tone}, or {audience}. When you insert the snippet, a quick dialog asks for the value — and it remembers your last input so you do not retype the same things.

✦ Pro plan ($9.99 one-time)
• Unlimited snippets (Free is limited to 5)
• Unlimited variables per snippet (Free supports 1)
• JSON import for migrating from existing libraries
• Categories and tags

✦ How to insert
1. Click the Prompt Snippets icon (or right-click any text field)
2. Pick a snippet from the list
3. The prompt is inserted directly into ChatGPT / Claude / Gemini / Perplexity
4. Or, if the page does not support direct insertion, the snippet is copied to your clipboard automatically

単一用途の説明
==============================
This extension has a single purpose: save reusable prompt templates and insert them into the input field of supported AI chat services (ChatGPT, Claude, Gemini, Perplexity) with one click.

==============================
storage が必要な理由
==============================
Required to save the user's prompt templates, categories, and variable history locally on the device using chrome.storage.local. No data is sent to any external server.

==============================
activeTab が必要な理由
==============================
Required to insert the selected prompt into the active tab's input field when the user explicitly clicks the extension icon, a snippet in the popup, or a context menu item. Combined with the "scripting" permission, this avoids requesting broad host permissions across many AI sites.

==============================
scripting が必要な理由
==============================
Required to inject the prompt-insertion code into the active tab only after the user explicitly chooses to insert a snippet. No code is injected automatically — only on user action.

==============================
contextMenus が必要な理由
==============================
Required to add a "Prompt Snippets" submenu to the right-click menu on editable text fields, allowing the user to insert any saved snippet directly from the context menu.

==============================
clipboardWrite が必要な理由
==============================
Used as a fallback when direct text insertion is not possible (e.g., on pages that block input modification). In this case, the snippet is copied to the clipboard so the user can paste it manually with Ctrl+V.

==============================
ホスト権限が必要な理由（content_scripts の matches）
==============================
https://extensionpay.com/*:
Required by ExtensionPay (the payment provider) to detect when a payment is completed. The ExtPay.js content script runs only on the ExtensionPay checkout page to communicate payment status back to the extension.

==============================
リモートコードを使用しているか
==============================
No. This extension does not use any remotely hosted code. All scripts (including ExtPay.js) are bundled within the extension package.
